## 7. Замечания для Wave 2

1. **Таблица 1 (8 критериев) = спецификация для гл.3.** Каждый критерий должен быть проверен на реализацию в System/.

2. **Когнитивный цикл (таблица 2, 8 этапов) = спецификация для кода.** Должен быть сопоставлен с компонентами System/.
