# 3.3.1 Техническая реализация инсталляции

В данном разделе описывается техническая реализация художественной инсталляции «Адам Чип» как физического воплощения нейроагента: материальное основание, аппаратная архитектура и интеграция вычислительного ядра с физической средой. Инсталляция реализует критерий воплощённости (2.1.7) на уровне физического присутствия агента.

## Материальная форма инсталляции

Инсталляция выполнена в форме **диорамы**: акриловое основание с грунтом, мхом, ветвями, корой и другими натуральными материалами образует экосистему — среду обитания «Адама Чипа». Физическое тело агента не отделено от среды: электронные компоненты — сенсоры, актуаторы, вычислительный модуль — интегрированы непосредственно в диораму, становясь частью единого организма.

Художественная концепция «цифрового симбионта» (раздел 3.1.1) воплощается через нерасчленённость электроники и природных материалов. Три типа «технофлоры» — элементов инсталляции с активным электронным поведением — образуют физический моторный слой агента (раздел 3.3.2).

## Аппаратная архитектура

Система построена на двух вычислительных узлах с чёткой специализацией:

**Вычислительный узел (Jetson)** — NVIDIA Jetson Orin NX Super 16 GB, выполняющий все инференс-операции: языковую модель, модель зрения, распознавание речи, синтез речи, оркестрацию. Jetson расположен вне видимой части инсталляции.

**Периферийный узел (ESP32)** — ESP32-S3 N16R8 WROOM CAM, управляющий сенсорами и актуаторами физического тела инсталляции. Узел обеспечивает: 16-канальный 12-bit PWM через PCA9685 (светофлора), управление звуком и вибрацией, захват MJPEG-видео с камеры OV5640, захват звука с двух микрофонов INMP441, Ethernet-подключение через W5500 SPI (статический IP 192.168.0.171).

Взаимодействие между узлами осуществляется по Ethernet HTTP/WebSocket. Jetson выступает клиентом (командует ESP32); ESP32 — сервером (принимает команды и отдаёт сенсорные данные).

## Полный состав аппаратного обеспечения

| Компонент | Модель | Назначение |
|-----------|--------|------------|
| Вычислительный модуль | Jetson Orin NX Super 16GB | Inference, orchestration |
| Периферийный МК | ESP32-S3 N16R8 WROOM CAM | Motor layer, sensors |
| ШИМ-расширитель | PCA9685 | 16×12bit PWM (light control) |
| Камера | OV5640 | MJPEG video stream |
| Микрофоны | 2× INMP441 stereo | Audio input |
| Аудио ЦАП | PCM5102A | High-quality audio output |
| Аудио усилитель | PAM8403 | Speaker amplification |
| Ethernet | W5500 SPI | Static IP network |
| Датчик освещённости | TEMT6000 | Ambient light sensing |
| Датчик пульса | BTE16-19 | Биометрический сенсор |

## Программное обеспечение инсталляции

Firmware ESP32-S3 разработана в среде PlatformIO и размещена в `Subsystem/AdamsServer/`. Прошивка обеспечивает HTTP API для приёма команд и WebSocket для push-телеметрии. Структура firmware детально описана в разделе 3.3.3.

На стороне Jetson управление инсталляцией осуществляется через `MCUClient` (`System/adam/device.py`) — HTTP-клиент с таймаутом 1 с и fire-and-forget семантикой. Конфигурация подключения: `mcu.base_url: http://192.168.0.171`, `mcu.speaker_url: http://192.168.0.171:81/speaker` (отдельный HTTP-сервер ESP32 для аудио, порт 81).

## Физическая воплощённость (критерий 2.1.7)

Реализованная инсталляция достигает **уровня физической воплощённости** (четвёртый уровень по критерию 2.1.7): агент не только обрабатывает информацию о физическом мире, но и воздействует на него через актуаторы, воспринимает через собственные сенсоры и существует в нём как физически расположенный объект. Светофлора, аудиофлора и виброфлора создают мультимодальное телесное присутствие.
