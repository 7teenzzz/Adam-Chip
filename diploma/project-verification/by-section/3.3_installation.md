## 3.3.1 Техническая реализация

**Verification.** Инсталляция = diorama (акрил + грунт + мох + ветви + кора + электроника).
- ✅ Jetson + ESP32 архитектура подтверждена README.
- ✅ Hardware spec: PCA9685, OV5640, INMP441, PCM5102A, PAM8403, W5500, TEMT6000, BTE16-19.
- ✅ Subsystem/AdamsServer/ содержит прошивку.

## 3.3.2 Перцептивный и моторный слои

**Verification.** Три типа технофлоры:
- ✅ Светофлора: PCA9685 PWM (16 channels × 12-bit) для LED.
- ✅ Аудиофлора: PCM5102A speaker (port 81 ESP32 server).
- ✅ Виброфлора: ESP32 GPIO для вибромоторов.

**Status:** **FULL**.

## 3.3.3 Программирование МК

**Verification.** ESP32-S3 firmware (Subsystem/AdamsServer/).
- ✅ Communication block (HTTP API)
- ✅ Motor block (3 channels: light/sound/vibration)
- ✅ Sensor block (camera, mic, ambient)
- ✅ Safety: idle scene fallback (`mcu.idle_scene: boot_idle`)
- ✅ Static IP (192.168.0.171) через W5500 SPI Ethernet.

**Status:** **FULL**.
