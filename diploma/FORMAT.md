Документ описывает требования к оформлению перед экспортом в .docx.

| Параметр          | Значение                                    |
| ----------------- | ------------------------------------------- |
| Размер            | 14 пт                                       |
| Начертание        | Полужирный                                  |
| Регистр           | С прописной буквы, остальные — строчные     |
| Выравнивание      | По ширине                                   |
| Отступ            | 1,25 см                                     |
| Точка в конце     | Нет                                         |
| Подчёркивание     | Нет                                         |

### 3.4. Общие правила для всех заголовков основной части

- Печатать с абзацного отступа 1,25 см, с прописной буквы, без точки в конце, без подчёркивания (остальные буквы строчные).
- Если заголовок состоит из двух предложений — разделять точкой.
- Если длинный заголовок переносится на следующую строку — начинать её также с отступа 1,25 см.
- Переносы слов внутри заголовков не допускаются.
- Интервал между заголовками разных уровней — 12 пт.
- Нельзя оставлять на предыдущей странице только заголовок без текста: в конце страницы должна быть хотя бы одна строка основного текста.

## 4. Интервалы между элементами

| Элемент                                              | Интервал до | Интервал после |
| ---------------------------------------------------- | ----------- | -------------- |
| Заголовок раздела → основной текст                   | —           | 12 пт          |
| Заголовок раздела → заголовок подраздела             | —           | 12 пт          |
| Рисунок (включая подпись) ↔ основной текст           | 12 пт       | 12 пт          |
| Формула ↔ основной текст                             | 12 пт       | 12 пт          |
| Таблица (включая название) ↔ основной текст          | 12 пт       | 12 пт          |

## 5. Кавычки

- Единственный допустимый вид кавычек — «ёлочки»: **«»**.
- Кавычки-лапки `""`, апострофные `''` и прямые `"` не допускаются нигде в тексте.
- Вложенные кавычки — «ёлочки» внутри «ёлочек»: «текст «вложение» текст».

---

## 6. Перечисления

- Перед каждым пунктом — тире (–), с абзацного отступа 1,25 см.
- В конце каждого пункта — точка с запятой (;).
- В конце последнего пункта — точка (.).

Пример:
```
– первый пункт перечисления;
– второй пункт перечисления;
– последний пункт перечисления.
```

---

## 7. Иллюстрации (рисунки)

- Нумерация сквозная арабскими цифрами (Рисунок 1, Рисунок 2, …).
- Подпись: слово «Рисунок», номер и наименование — **после** рисунка, по центру строки.
  - Пример: `Рисунок 3 — Архитектура системы`
- Если наименование занимает несколько строк — записывать через один межстрочный интервал.
- На каждую иллюстрацию должна быть ссылка в тексте: «…как показано на рисунке 3».
- Рисунок располагается непосредственно под текстом, в котором впервые дана ссылка на него.

---

## 8. Таблицы

- Располагается на всю ширину печатного поля.
- Помещается под текстом первой ссылки или на следующей странице.
- Название — над таблицей, **без абзацного отступа**, слева, формат:
  - `Таблица <Номер> — <Наименование>`
  - Пример: `Таблица 2 — Параметры конфигурации`
- Наименование — с прописной буквы, без точки в конце.
- Если наименование занимает две строки и более — записывать через один межстрочный интервал.
- Нумерация сквозная.

| Тип элемента     | Формат ссылки                        |
| ---------------- | ------------------------------------ |

## 10. Нумерация страниц

- Сквозная, арабскими цифрами.
- Размещение: нижний колонтитул, по центру.
- Титульный лист включается в нумерацию, но номер на нём не проставляется.
